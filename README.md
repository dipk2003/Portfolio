# 🌟 Personal Portfolio Website

A modern, responsive personal portfolio website showcasing skills, projects, and professional experience. Built with HTML5, CSS3, and JavaScript with smooth animations.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

---

## ✨ Features

- **Responsive Design** — Optimized for mobile, tablet, and desktop with multiple breakpoints
- **Typing Animation** — Dynamic hero section with Typed.js effects
- **Smooth Scrolling** — Fixed navbar with smooth scroll navigation
- **Project Showcase** — Interactive project cards with hover effects
- **Skills Visualization** — Animated progress bars for technical skills
- **Resume Download** — One-click PDF resume download
- **Contact Form** — Functional form with Netlify integration
- **Scroll-to-Top** — Quick navigation button
- **Mobile Menu** — Hamburger menu for small screens

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Semantic markup |
| CSS3 | Custom styling with Flexbox and Grid |
| JavaScript | Interactive functionality |
| jQuery 3.5.1 | DOM manipulation |
| Typed.js 2.0.11 | Typing animations |
| Owl Carousel 2.3.4 | Responsive carousel |
| Font Awesome 5.15.3 | Icons |
| Google Fonts | Typography (Poppins, Ubuntu) |
| Netlify | Hosting and form handling |

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/dipk2003/Portfolio.git
cd Portfolio

# Open in browser
open index.html

# Or use a live server
npx serve .
```
